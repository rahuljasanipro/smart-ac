🐛 IF SOMETHING IS BROKEN:
─────────────────────────────────────────────────────────────────────────────

Step 1: Check file names
  → MUST be: index.html, style.css, script.js
  → NO "advanced-" prefix
  → All in SAME folder

Step 2: Hard refresh browser
  → Windows: Ctrl+F5
  → Mac: Cmd+Shift+R
  → Or: Clear cache (Ctrl+Shift+Delete)

Step 3: Check browser console
  → Press F12
  → Look at Console tab
  → Fix any red errors shown

Step 4: Try different browser
  → Chrome, Firefox, Safari, Edge
  → All should work

Step 5: Check internet connection
  → Charts need CDN access
  → Other features work offline

See SETUP.md for detailed troubleshooting!
