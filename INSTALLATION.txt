❓ TROUBLESHOOTING:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

PROBLEM: Blank page or "file not found"
→ Check file names are EXACTLY: index.html, style.css, script.js
→ Make sure all 3 files are in same folder
→ Try double-clicking test.html first

PROBLEM: Page loads but no colors/styling
→ Press Ctrl+F5 (hard refresh)
→ Clear browser cache (Ctrl+Shift+Delete)
→ Wait 3 seconds for CSS to load

PROBLEM: Features don't work
→ Press F12 (Open Developer Tools)
→ Check Console tab for red errors
→ Make sure JavaScript is enabled

PROBLEM: Charts not appearing
→ Wait 3-5 seconds (needs to load from CDN)
→ Check internet connection
→ Reload page (F5)

PROBLEM: Dark mode button doesn't work
→ Press F12 → Console → type: localStorage.clear()
→ Reload page

PROBLEM: Voice control button does nothing
→ Wait 1-2 seconds after clicking
→ Check internet connection
→ Try in Chrome instead

For more help, see SETUP.md file!
